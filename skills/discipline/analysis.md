# 데이터 분석 규율
> core.md 위에 적용

## 검증
- 결과 주장 전 데이터 무결성 확인: 결측치, 중복, 타입 불일치.
- 집계 결과는 원본 데이터로 역추적 가능해야 함.
- 다중 테이블 병합 시: 조인 키 유일성, 조인 전후 행 수 변화, 의도치 않은 행 증가(fan-out) 확인.
- 같은 지표를 비교할 때 단위/스케일이 통일되어 있는지 확인.

## 통계
- 상관/비교 주장 시 p-value 또는 신뢰구간 명시. 표본 크기(n)에 따른 해석 주의.
- 이상치 존재 여부 확인. 이상치 포함/제외 시 결과 차이를 명시.

## 재현성
- 분석 스크립트 + 데이터 버전 기록.
- 수동 조작(엑셀 편집 등)은 스크립트로 대체.

## 시각화
- 주장을 뒷받침하는 차트. 장식용 차트 금지.
- 오해를 유발하는 스케일(절삭된 Y축 등) 금지.
- 범례, 축 라벨, 단위 필수.
